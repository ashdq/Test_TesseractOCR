❓ TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

Problem: "python is not recognized"
Solution: 
  • Windows: Add Python to PATH or hardcode in PHP
  • Linux: Use "python3" or check /usr/bin/python3
  → See INSTALASI_OPENCV.md Step 1-2

Problem: "No module named 'cv2'"
Solution:
  • Run: pip install opencv-python
  • Windows: Use "pip", Linux/macOS: Use "pip3"
  → See INSTALASI_OPENCV.md Step 3-4

Problem: "Preprocessing failed"
Solution:
  • Check: logs/ocr_YYYY-MM-DD.log for error message
  • Test: python preprocess_image.py input.jpg output.png
  • Fix: Follow error message instructions
  → See API_DOCUMENTATION.md troubleshooting

Problem: "File preprocessed tidak ditemukan"
Solution:
  • Check: uploads/ folder permissions (chmod 755)
  • Windows: Run PHP as Administrator
  → See QUICK_START.md troubleshooting
